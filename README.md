# Reinforcement Learning Agents

A modular reinforcement learning framework implementing REINFORCE and PPO agents.  
Supports both discrete and continuous action spaces, using PyTorch.

## Usage

```bash
poetry install
poetry run python envs/lunar/lander.py
```

## Implemented

- [x] Reinforce Agent
- [ ] PPO Agent

## Dependencies

- Python 3.10+
- PyTorch
- Gymnasium
